# 基于Java的社团管理系统 java1022

## 项目概述

本项目是基于Java的社团管理系统，采用SSM（Spring、SpringMVC、Mybatis）框架进行开发，结合JSP、CSS、HTML、JavaScript等前端技术，实现了Web形式的社团管理功能。

## 技术栈

### 后端技术
- Java
- MySQL
- Spring
- SpringMVC
- Mybatis

### 前端技术
- JSP
- CSS
- HTML
- JavaScript

### 开发及运行环境
- 系统环境：Windows/Mac
- 开发环境：JDK 1.8、Maven、MySQL、Tomcat
- 数据库：MySQL 5.7（推荐）
- 开发工具：Eclipse/MyEclipse/IDEA
- Maven版本：Maven 3.3.9
- 推荐浏览器：谷歌浏览器

## 功能模块

系统功能分为管理员、社长和普通用户三个角色，主要功能模块包括：

- 个人中心
- 用户管理
- 社长管理
- 社团类别管理
- 社团信息管理
- 活动类型管理
- 社团活动管理
- 入团申请管理
- 退团申请管理
- 在线留言
- 系统管理

## 系统角色

1. **用户**
   - 管理个人中心
   - 管理入团、退团申请

2. **社长**
   - 管理个人中心
   - 管理社团活动
   - 管理入团、退团申请

3. **管理员**
   - 添加和管理用户信息
   - 添加和管理社团详细信息
   - 添加和管理文档信息
   - 管理系统信息

## 核心亮点

- 基于B/S架构，便于管理和使用
- 采用MVC三层体系结构，使系统逻辑清晰
- 前后端不分离，适合传统Web应用开发
- 界面简洁，易于操作

## 部署步骤

1. 准备开发环境：JDK 1.8、MySQL 5.7、Maven 3.3.9等
2. 克隆或解压源码到本地工作目录
3. 使用Maven构建项目，确保依赖正确解析
4. 导入数据库文件，创建所需数据库和表
5. 部署至Tomcat服务器
6. 根据配置文件调整相关设置
7. 运行项目，开始使用
